# 🛒 Hudders Hub - E-Commerce Website

Hudders Hub is a simple and modern PHP-based e-commerce web application designed for showcasing and selling local products online. The project is built using PHP, MySQL, HTML, CSS, and JavaScript, and runs on a local XAMPP server.

---

## 🚀 Features

- 🏠 Modern homepage UI
- 🛍️ Product listing system
- 🧾 Product detail pages
- 🛒 Add to cart functionality (basic structure)
- 📦 Inventory-ready structure
- 🎨 Responsive UI design
- 🔐 Admin-ready backend structure (expandable)
- 🗂️ Organized file structure for scalability

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP
- **Database:** MySQL (optional / extendable)
- **Server:** XAMPP (Apache + MySQL)
